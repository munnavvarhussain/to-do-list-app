# To-Do List Web App

This is a front-end To-Do List application built using **HTML, CSS, and Vanilla JavaScript**.

## Features
- Add and remove tasks
- Mark tasks as complete
- Responsive design
- Dark mode toggle
- Task counter
- Input validation
